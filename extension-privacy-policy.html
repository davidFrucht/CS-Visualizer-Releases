CS Visualizer Browser Extension — Privacy Policy

Last updated: April 4, 2026

Overview

The CS Visualizer browser extension detects code blocks on webpages and allows you to open them directly in the CS Visualizer desktop application running locally on your machine.

What the Extension Does

  • Scans pages for code blocks (<pre> / <code> elements) and injects an "Open in CS Visualizer" button next to each one.
  • When you click the button, the code text from that block is passed to the CS Visualizer desktop application running on your own machine via localhost.
  • No code or page content is ever sent to any external or remote server.

Data Stored

The extension stores a single on/off preference (csv_ext_enabled) in your browser's local storage. This data never leaves your device.

Local Communication

The extension communicates exclusively with the CS Visualizer desktop application via localhost:8080. All data stays on your device.

Desktop Application

The CS Visualizer desktop application that this extension connects to requires a user account for authentication and license verification (via Google Firebase). Please refer to the CS Visualizer full privacy policy for details on how the desktop application handles your data.

Third-Party Services

The extension does not integrate with any third-party analytics, advertising, or tracking services.

Children's Privacy

The extension does not collect any information from any user, including children under 13.

Changes to This Policy

Any updates to this policy will be posted on this page with an updated date.
